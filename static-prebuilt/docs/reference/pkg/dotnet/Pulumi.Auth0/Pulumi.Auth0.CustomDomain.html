Class CustomDomain

With Auth0, you can use a custom domain to maintain a consistent user experience. This resource allows you to create and manage a custom domain within your Auth0 tenant.

Example Usage

using Pulumi;
using Auth0 = Pulumi.Auth0;

class MyStack : Stack
{
public MyStack()
{
    var myCustomDomain = new Auth0.CustomDomain(&quot;myCustomDomain&quot;, new Auth0.CustomDomainArgs
    {
        Domain = &quot;auth.example.com&quot;,
        Type = &quot;auth0_managed_certs&quot;,
        VerificationMethod = &quot;txt&quot;,
    });
}

}
