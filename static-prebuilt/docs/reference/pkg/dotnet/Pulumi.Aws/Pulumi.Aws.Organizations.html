Account

Provides a resource to create a member account in the current organization.

Note: Account management must be done from the organization's master account.

!> WARNING: Deleting this resource will only remove an AWS account from an organization. This provider will not close the account. The member account must be prepared to be a standalone account beforehand. See the AWS Organizations documentation for more information.

Example Usage

using Pulumi;
using Aws = Pulumi.Aws;

class MyStack : Stack
{
public MyStack()
{
    var account = new Aws.Organizations.Account(&quot;account&quot;, new Aws.Organizations.AccountArgs
    {
        Email = &quot;john@doe.org&quot;,
    });
}

}

Organization

Provides a resource to create an organization.

Example Usage

using Pulumi;
using Aws = Pulumi.Aws;

class MyStack : Stack
{
public MyStack()
{
    var org = new Aws.Organizations.Organization(&quot;org&quot;, new Aws.Organizations.OrganizationArgs
    {
        AwsServiceAccessPrincipals = 
        {
            &quot;cloudtrail.amazonaws.com&quot;,
            &quot;config.amazonaws.com&quot;,
        },
        FeatureSet = &quot;ALL&quot;,
    });
}

}
