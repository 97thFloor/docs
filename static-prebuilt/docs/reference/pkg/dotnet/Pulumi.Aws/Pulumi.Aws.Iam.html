AccessKey

Provides an IAM access key. This is a set of credentials that allow API requests to be made as an IAM user.

Example Usage

using Pulumi;
using Aws = Pulumi.Aws;

class MyStack : Stack
{
public MyStack()
{
    var lbUser = new Aws.Iam.User(&quot;lbUser&quot;, new Aws.Iam.UserArgs
    {
        Path = &quot;/system/&quot;,
    });
    var lbAccessKey = new Aws.Iam.AccessKey(&quot;lbAccessKey&quot;, new Aws.Iam.AccessKeyArgs
    {
        PgpKey = &quot;keybase:some_person_that_exists&quot;,
        User = lbUser.Name,
    });
    var lbRo = new Aws.Iam.UserPolicy(&quot;lbRo&quot;, new Aws.Iam.UserPolicyArgs
    {
        Policy = @&quot;{
&quot;&quot;Version&quot;&quot;: &quot;&quot;2012-10-17&quot;&quot;,
&quot;&quot;Statement&quot;&quot;: [
{
  &quot;&quot;Action&quot;&quot;: [
    &quot;&quot;ec2:Describe*&quot;&quot;
  ],
  &quot;&quot;Effect&quot;&quot;: &quot;&quot;Allow&quot;&quot;,
  &quot;&quot;Resource&quot;&quot;: &quot;&quot;*&quot;&quot;
}
]
}

&quot;,
        User = lbUser.Name,
    });
    this.Secret = lbAccessKey.EncryptedSecret;
}

WARNING: Multiple aws.iam.GroupMembership resources with the same group name will produce inconsistent behavior!

Provides a top level resource to manage IAM Group membership for IAM Users. For more information on managing IAM Groups or IAM Users, see IAM Groups or IAM Users

PolicyAttachment

Attaches a Managed IAM Policy to user(s), role(s), and/or group(s)

!> WARNING: The aws.iam.PolicyAttachment resource creates exclusive attachments of IAM policies. Across the entire AWS account, all of the users/roles/groups to which a single policy is attached must be declared by a single aws.iam.PolicyAttachment resource. This means that even any users/roles/groups that have the attached policy via any other mechanism (including other resources managed by this provider) will have that attached policy revoked by this resource. Consider aws.iam.RolePolicyAttachment, aws.iam.UserPolicyAttachment, or aws.iam.GroupPolicyAttachment instead. These resources do not enforce exclusive attachment of an IAM policy.

NOTE: The usage of this resource conflicts with the aws.iam.GroupPolicyAttachment, aws.iam.RolePolicyAttachment, and aws.iam.UserPolicyAttachment resources and will permanently show a difference if both are defined.

Example Usage

using Pulumi;
using Aws = Pulumi.Aws;

class MyStack : Stack
{
public MyStack()
{
    var user = new Aws.Iam.User(&quot;user&quot;, new Aws.Iam.UserArgs
    {
    });
    var role = new Aws.Iam.Role(&quot;role&quot;, new Aws.Iam.RoleArgs
    {
        AssumeRolePolicy = @&quot;{
&quot;&quot;Version&quot;&quot;: &quot;&quot;2012-10-17&quot;&quot;,
&quot;&quot;Statement&quot;&quot;: [
{
  &quot;&quot;Action&quot;&quot;: &quot;&quot;sts:AssumeRole&quot;&quot;,
  &quot;&quot;Principal&quot;&quot;: {
    &quot;&quot;Service&quot;&quot;: &quot;&quot;ec2.amazonaws.com&quot;&quot;
  },
  &quot;&quot;Effect&quot;&quot;: &quot;&quot;Allow&quot;&quot;,
  &quot;&quot;Sid&quot;&quot;: &quot;&quot;&quot;&quot;
}
]
}

&quot;,
    });
    var @group = new Aws.Iam.Group(&quot;group&quot;, new Aws.Iam.GroupArgs
    {
    });
    var policy = new Aws.Iam.Policy(&quot;policy&quot;, new Aws.Iam.PolicyArgs
    {
        Description = &quot;A test policy&quot;,
        Policy = @&quot;{
&quot;&quot;Version&quot;&quot;: &quot;&quot;2012-10-17&quot;&quot;,
&quot;&quot;Statement&quot;&quot;: [
{
  &quot;&quot;Action&quot;&quot;: [
    &quot;&quot;ec2:Describe*&quot;&quot;
  ],
  &quot;&quot;Effect&quot;&quot;: &quot;&quot;Allow&quot;&quot;,
  &quot;&quot;Resource&quot;&quot;: &quot;&quot;*&quot;&quot;
}
]
}

&quot;,
    });
    var test_attach = new Aws.Iam.PolicyAttachment(&quot;test-attach&quot;, new Aws.Iam.PolicyAttachmentArgs
    {
        Groups = 
        {
            @group.Name,
        },
        PolicyArn = policy.Arn,
        Roles = 
        {
            role.Name,
        },
        Users = 
        {
            user.Name,
        },
    });
}

UserLoginProfile

Manages an IAM User Login Profile with limited support for password creation during this provider resource creation. Uses PGP to encrypt the password for safe transport to the user. PGP keys can be obtained from Keybase.

To reset an IAM User login password via this provider, you can use delete and recreate this resource or change any of the arguments.

using Aws = Pulumi.Aws;

public MyStack()

    {
        ForceDestroy = true,
