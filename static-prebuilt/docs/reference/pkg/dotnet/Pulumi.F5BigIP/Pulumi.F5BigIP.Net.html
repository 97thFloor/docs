Route

f5bigip.net.Route Manages a route configuration

For resources should be named with their "full path". The full path is the combination of the partition + name of the resource. For example /Common/my-pool.

Example Usage

using Pulumi;
using F5BigIP = Pulumi.F5BigIP;

class MyStack : Stack
{
public MyStack()
{
    var route2 = new F5BigIP.Net.Route(&quot;route2&quot;, new F5BigIP.Net.RouteArgs
    {
        Gw = &quot;1.1.1.2&quot;,
        Name = &quot;external-route&quot;,
        Network = &quot;10.10.10.0/24&quot;,
    });
}

}
