Provider

The provider type for the fastly package. By default, resources use package-wide configuration settings, however an explicit Provider instance may be created and passed during resource construction to achieve fine-grained programmatic control over provider settings. Servicev1

Provides a Fastly Service, representing the configuration for a website, app, API, or anything else to be served through Fastly. A Service encompasses Domains and Backends.

The Service resource requires a domain name that is correctly set up to direct traffic to the Fastly service. See Fastly's guide on [Adding CNAME Records][fastly-cname] on their documentation site for guidance.

Example Usage

Basic usage

using Pulumi;
using Fastly = Pulumi.Fastly;

class MyStack : Stack
{
public MyStack()
{
    var demo = new Fastly.Servicev1(&quot;demo&quot;, new Fastly.Servicev1Args
    {
        Backends = 
        {
            new Fastly.Inputs.Servicev1BackendArgs
            {
                Address = &quot;127.0.0.1&quot;,
                Name = &quot;localhost&quot;,
                Port = 80,
            },
        },
        Domains = 
        {
            new Fastly.Inputs.Servicev1DomainArgs
            {
                Comment = &quot;demo&quot;,
                Name = &quot;demo.notexample.com&quot;,
            },
        },
        ForceDestroy = true,
    });
}

}

Basic usage with custom VCL:

using System.IO;
using Pulumi;
using Fastly = Pulumi.Fastly;

class MyStack : Stack
{
public MyStack()
{
    var demo = new Fastly.Servicev1(&quot;demo&quot;, new Fastly.Servicev1Args
    {
        Backends = 
        {
            new Fastly.Inputs.Servicev1BackendArgs
            {
                Address = &quot;127.0.0.1&quot;,
                Name = &quot;localhost&quot;,
                Port = 80,
            },
        },
        Domains = 
        {
            new Fastly.Inputs.Servicev1DomainArgs
            {
                Comment = &quot;demo&quot;,
                Name = &quot;demo.notexample.com&quot;,
            },
        },
        ForceDestroy = true,
        Vcls = 
        {
            new Fastly.Inputs.Servicev1VclArgs
            {
                Content = File.ReadAllText($&quot;{path.Module}/my_custom_main.vcl&quot;),
                Main = true,
                Name = &quot;my_custom_main_vcl&quot;,
            },
            new Fastly.Inputs.Servicev1VclArgs
            {
                Content = File.ReadAllText($&quot;{path.Module}/my_custom_library.vcl&quot;),
                Name = &quot;my_custom_library_vcl&quot;,
            },
        },
    });
}

}

            new Fastly.Inputs.Servicev1DirectorArgs

                Name = &quot;mydirector&quot;,
                Quorum = 0,
                Type = 3,
            },
        },
        Domains = 
        {
            new Fastly.Inputs.Servicev1DomainArgs
            {
                Comment = &quot;demo&quot;,
                Name = &quot;demo.notexample.com&quot;,
            },
        },
        ForceDestroy = true,
    });

Userv1

Provides a Fastly User, representing the configuration for a user account for interacting with Fastly.

The User resource requires a login and name, and optionally a role.

Example Usage

using Pulumi;
using Fastly = Pulumi.Fastly;

class MyStack : Stack
{
public MyStack()
{
    var demo = new Fastly.Userv1(&quot;demo&quot;, new Fastly.Userv1Args
    {
        Login = &quot;demo@example.com&quot;,
    });
}

}
