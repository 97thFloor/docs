Class License

NOTE: This feature requires Consul Enterprise.

The consul..License resource provides datacenter-level management of the Consul Enterprise license. If ACLs are enabled then a token with operator privileges may be required in order to use this command.

Example Usage

using System.IO;
using Pulumi;
using Consul = Pulumi.Consul;

class MyStack : Stack
{
public MyStack()
{
    var license = new Consul.License(&quot;license&quot;, new Consul.LicenseArgs
    {
        License = File.ReadAllText(&quot;license.hclic&quot;),
    });
}

}
