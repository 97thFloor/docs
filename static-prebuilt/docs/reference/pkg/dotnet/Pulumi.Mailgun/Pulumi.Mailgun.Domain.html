Class Domain

Provides a Mailgun App resource. This can be used to create and manage applications on Mailgun.

After DNS records are set, domain verification should be triggered manually using PUT /domains/&lt;domain>/verify

Example Usage

using Pulumi;
using Mailgun = Pulumi.Mailgun;

class MyStack : Stack
{
public MyStack()
{
    // Create a new Mailgun domain
    var @default = new Mailgun.Domain(&quot;default&quot;, new Mailgun.DomainArgs
    {
        Region = &quot;us&quot;,
        SpamAction = &quot;disabled&quot;,
    });
}

}
