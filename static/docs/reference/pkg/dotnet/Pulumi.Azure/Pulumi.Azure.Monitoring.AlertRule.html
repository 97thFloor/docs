Class AlertRule

Manages a metric-based alert rule in Azure Monitor.

NOTE: This resource has been deprecated in favour of the azure.monitoring.MetricAlert resource and will be removed in the next major version of the AzureRM Provider. The new resource shares the same fields as this one, and information on migrating across can be found in this guide.
